# HLSMP-TPA

Plugin TPA cho máy chủ HLSMP — giao diện GUI xác nhận yêu cầu dịch chuyển, hỗ trợ đầy đủ Folia.

## Lệnh

| Lệnh | Mô tả |
|------|-------|
| `/tpa <người chơi>` | Gửi yêu cầu TPA |
| `/tpahere <người chơi>` | Gửi yêu cầu TPA Here |
| `/tpaccept` | Chấp nhận yêu cầu TPA |
| `/tpadeny` | Từ chối yêu cầu TPA |
| `/tpacancel` | Huỷ yêu cầu TPA đã gửi |
| `/tpatoggle` | Bật/tắt nhận yêu cầu TPA |
| `/tpaheretoggle` | Bật/tắt nhận yêu cầu TPA Here |
| `/tpagui` / `/tpaconfirmtoggle` | Bật/tắt GUI xác nhận TPA |
| `/tpaheregui` | Bật/tắt GUI xác nhận TPA Here |
| `/tpauto` | Bật/tắt tự động chấp nhận TPA |
| `/tpareload` | Tải lại cấu hình (yêu cầu `tpa.admin`) |

## Quyền

| Quyền | Mặc định |
|-------|----------|
| `tpa.admin` | OP |

## Cấu hình

Chỉnh sửa `config.yml` để thay đổi thời gian chờ, độ trễ teleport, âm thanh và trạng thái bay.
